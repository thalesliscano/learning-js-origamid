Funções é um bloco de codigo que podem ser executas e reutilizados.


    function areaQuadrado(lado) {
        return lado * lado;
    }

    areaQuadrado(4) // 16
    areaQuadrado(5) // 25
    areaQuadrado(2) // 4

Ao Criar funções, você pode definir argumentos. E ao executar funções você pode passar argumentos


    // peso e altura são os parâmetros
    function imc(peso, altura) {
        const imc = peso / (altura ** 2);
        return imc;
    }

    imc(80, 1.80) // 80 e 1.80 são os argumentos
    imc(60, 1.70) // 60 e 1.70 são os argumentos

Parêntes executam as funções.


    function saudacao(nome) {
        console.log("Olá, " + nome + "!");
    }

    saudacao("Mundo");

Argumentos podem ser funções


    addEventListener('click', function() {
        console.log('Clicou');
    });
    // A função possui dois argumentos. Primeiro é a string 'click'. Segundo é uma função anônima

Podem retornar ou não um valor

Uma função pode retornar qualquer tipo de dado e até outras funções.
